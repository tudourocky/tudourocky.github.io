Control Statements

If statement

if the condition given in the if function is ture, it will run the code within this condition.

if the condition given is false, the code within the function will be skipped.

1
2
3
4
5

Boolean a = true, b = false;

if(a){ // if a is true, keep going.
  System.out.println("Hello");
}

import java.util.ArrayList; //Import to use list
import java.util.List;

List<String> list = new ArrayList<String>();
list.add("Deer");
list.add("Duck");
list.add("Buck");

for(String s : list){ //for each string in string s in list.
//s is a variable that I defined.
  System.out.println(s); // print out the list.
}

2D array for loop

int[][] arry = {{1, 2, 3},{1, 2, 3},{1, 2, 3}};

for(int row = 0; row < arry.length; row ++){
	for(int col = 0; col < arry[row].length; col++){
		System.out.print(arry[row][col] + " ");
	}
	System.out.println();
}
