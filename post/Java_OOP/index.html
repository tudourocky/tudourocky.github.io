Difference between private class and public class

public means you can access it anywhere while private means you can only access it inside its own class.

Just to note all private, protected or public modifier are not applicable to local variables in Java. a local variable can only be final in java.

package VideoProj;

public abstract class Human { //abstract classes can not be instantiated
	private String name;
	private int age;

	public Human() { // constructors
		name = "";
		age = 0;
	}

	public Human(String nm, int ag) { // constructors
		name = nm;
		age = ag;
	}

	public String getName() { // accessor
		return name;
	}

	public int getAge() { // accessor access a value
		return age;
	}

	public abstract int ageInDogYears(); // abstract method

	public void setName(String s) { // mutator changes a state of an object
		name = s;
	}
	//override
	public String toString(){
		return "Human: " + name + ", Age: " + age;
	}

}

package VideoProj;

public class Athlete extends Human{ //the extends Human   makes Athlete a subclass of the class human
	private String sport;

	public Athlete() { // constructor
  //		super();
  //super must be used on the first line. References the super class which is human in this case
		sport = "";
	}

	public Athlete(String nm, int ag, String s) {
		super(nm, ag);
		sport = s;
	}

	public String getSport() {
		return sport;
	}

	public int ageInDogYears() {
		return super.getAge() * 7;
	}

	public void setSport(String s) {
		sport = s;
	}

	public String toString() {
		return "Athlete: " + super.getName() + ", " + sport;
	}
}

package VideoProj;

public class Athlete implements Person { //the extends Human makes Athlete a subclass of the class human
	private String sport;
	private String name;
	private int age;
	private static int count = 0;
	public final static int var = 500; // final makes this integer unchangeable

	public Athlete() { // constructor
		sport = "";
		count++;
	}

	public Athlete(String nm, int ag, String s) {

		sport = s;
	}

	public static int getCount() {
		return count;
	}


	public String getName() {
		return name;
	}

	public int getAge() {
		return age;
	}

	public void setName(String s) {
		name = s;
	}

	public void setAge(int i) {
		age = i;
	}
	public String getSport() {
		return sport;
	}

	public void setSport(String s) {
		sport = s;
	}

	public String toString() {
		return "Athlete: " + name + ", " + sport;
	}
}

Equals method

  1. The equals method is different from ==.

  2. == compares tow objects to see if they exists in the same place in memory.

  3. .equals method compares two strings to see if they are exactly equal.
