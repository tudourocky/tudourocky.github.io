# PortfolioV2

A minimalist, terminal-inspired personal portfolio website.

## Features

- Terminal/CLI aesthetic with command-prompt styling
- Dark and light theme toggle with localStorage persistence
- Staggered fade-in animations on load
- Fully responsive design
- Zero dependencies (vanilla HTML, CSS, JS)

## Getting Started

Open `index.html` in a browser, or serve it with any static file server:

```bash
npx serve .
```

## Customization

Edit `index.html` to update:

- Your name and tagline
- Contact links (GitHub, LinkedIn, email)
- Recent work/experience entries
- About section text
